# 场景拆分 — 极端构造测试样本

三个**刻意构造**的压力测试样本,验证场景拆分在结构极端时的上下界表现
(该1段的别拆碎、该多段的别封顶)。配合 test_chapters/ 的真实章节一起用。

| 样本 | 人工预期段数 | 设计意图 |
|---|---|---|
| A 单一动作 | **1** | 纯粹一连串连贯动作,无功能转折→测"该合的别拆碎" |
| B 短对话 | **1** | 极短单场景对话→测下界(别被"至少几段"推着过度拆) |
| C 多跳转 | **6** | 6次时空跳转(现实→回忆→内心→现实→内心→动作)挤在一段→测上界(别封顶) |

注:v8 定稿后,C 的内心独白层不独立成段(下放事件层),实际稳定输出按 loc+回忆time 边界计。

---

## A_单一动作 (预期1段)

```
他走进厨房，拧开水龙头，接了半壶水，放到灶台上。点着火，蓝色的火苗舔着壶底。他从橱柜里取出茶叶罐，拧开盖子，舀了两勺茶叶放进杯里。水开了，他提起壶，缓缓注入沸水，茶叶在水中翻滚舒展。他端起杯子，吹了吹热气，走到窗边坐下，望着楼下的街道，慢慢啜饮。
```

## B_短对话 (预期1段)

```
"你来了。"老张抬起头。"嗯。"我点点头，在他对面坐下。"事情办妥了？"他问。"办妥了。"我说。他笑了笑，没再说话。
```

## C_多跳转 (预期6段;v8下按硬边界计)

```
林深站在码头上，海风卷着咸腥扑面而来。货轮的汽笛长鸣，惊起一群海鸥。

三年前的那个雨夜又浮现在眼前。他和苏雯就是在这个码头分别的，她撑着伞回头看了他最后一眼，雨水顺着伞沿滴落。那时他还不知道，那竟是永别。

"如果当初我留下来就好了。"林深握紧了栏杆，指节发白。早知如此，他宁愿放弃那个所谓的机会，也要守在她身边。可惜世上没有如果。

汽笛再次响起，把他拉回现实。搬运工们正喊着号子卸货，箱子在他们肩头摞起又放下。一个工头模样的人朝他走来。

他忽然想起昨天收到的那封信。信封是苏雯的笔迹，可她明明已经……他不敢再想下去。

林深深吸一口气，转身大步朝仓库走去。他推开沉重的铁门，穿过堆满货物的过道，在最里间的办公室前停下，抬手敲了三下门。
```
