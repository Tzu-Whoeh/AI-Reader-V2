# test_chapters/ — 测试章节基准

研发中反复使用的真实章节,各有不同结构特征,覆盖不同测试需求。改任何 pass 时用这几章做回归。

| 文件 | 字数 | 内容 | 结构特征 | 适合测什么 |
|---|---|---|---|---|
| `ch44.txt` | ~3.9k | 华剑雄在办公室质问柳媚→亲密→独处沉思→思绪转何小月案 | **单一地点(办公室)**,现实对话与内心独白高频穿插,结尾思绪跳别处 | 场景:同地点内的高频功能切换(易过度细切);人物:柳媚多别名 |
| `doc3.txt` | ~2.7k | 办公室接电话→得胜楼与丁墨村密谈→专车上看报惊闻遇刺 | **多地点跳转(3场景)**,大量对话 + 结尾时空层切换 | 场景:loc 硬边界(最稳);事件:多场景子事件分组;人物:"提及vs在场"(丁墨村被转述) |
| `doc4.txt` | ~2.2k | 霞露公寓地下党争论营救→众人离去→周雪萍独哭→大段往事回放 | **单地点 + 人物分批离场(cast) + 大段往事夹叙** | 场景:cast软边界、往事回放(time);人物:**周丽萍/周雪萍姐妹易误合** |
| `doc5.txt` | ~3.0k | 周老板办公室召见华剑雄,谈长春刺杀案、交代审案、配秘书 | **单一地点单场连续长对话**,话题多次推进但场次不断 | 场景:验证"话题推进不该拆"(单场景对话→应1段);人物:何小月被提及 |

## 已知"陷阱"(各章会暴露的抽取错,记录备查)

- **ch44**:人物 prompt 易把"老段落"黏连成人名(应为"老段");物品 prompt 易把"医生"当物品、
  "特别囚室"当物品又当地点、"藤原香子的榻榻米"带前缀。
- **doc4**:人物 prompt 易把"周雪萍"误作"周丽萍"的别名(姐妹两人,参与不同事件,不可合并)。
- **doc3**:事件抽取易把转述提及的"丁墨村"(电话那头,不在场)误列为电话场景参与者。
  → 自由参与者 prompt(直接读原文人名 + "提及≠参与"规则)可修正此错。
